# 프로젝트 이미지 추가 가이드

## 📸 이미지 추가 방법

각 프로젝트에 대표 이미지를 추가할 수 있습니다.

### 1. 이미지 파일 준비

프로젝트 대표 이미지 파일을 준비합니다:
- 권장 형식: JPG, PNG, WebP
- 권장 크기: 1200px × 800px (16:9 비율)
- 파일 크기: 500KB 이하 권장 (빠른 로딩을 위해)

### 2. 이미지 파일 저장 위치

이미지 파일을 다음 경로에 저장하세요:

```
public/
  images/
    projects/
      project1.jpg  (쓰담쓰담 프로젝트)
      project2.jpg  (스마트팜 프로젝트)
      project3.jpg  (mong 프로젝트)
      project4.jpg  (Vitamin 7 프로젝트)
```

### 3. 파일 이름 규칙

각 프로젝트에 맞는 파일명:
- `project1.jpg` - 쓰담쓰담 앱/웹
- `project2.jpg` - 스마트팜 모니터링
- `project3.jpg` - mong (몽) 수면 분석 플랫폼
- `project4.jpg` - 카카오워크 챗봇 관리 플랫폼

또는 원하는 파일명을 사용하고 `Projects.jsx` 파일에서 경로를 수정하세요.

### 4. 이미지가 없을 때

이미지를 추가하지 않으면 자동으로 그라데이션 배경에 프로젝트 번호가 표시됩니다.

## 🖼️ 이미지 경로 변경하기

`src/components/Projects.jsx` 파일에서 각 프로젝트의 `image` 속성을 수정하세요:

```javascript
{
  id: 1,
  title: '쓰담쓰담 앱/웹',
  // ... 기타 속성들
  image: '/images/projects/project1.jpg', // 여기를 수정
}
```

## 💡 이미지 최적화 팁

1. **온라인 이미지 압축 도구 사용**
   - TinyPNG: https://tinypng.com/
   - Squoosh: https://squoosh.app/

2. **반응형 이미지**
   - 현재는 하나의 이미지를 사용하며, 자동으로 크기가 조정됩니다.
   - 필요시 더 큰 해상도의 이미지를 사용해도 자동으로 최적화됩니다.

3. **WebP 형식 사용**
   - 최신 브라우저에서 더 작은 파일 크기로 고품질 이미지를 제공합니다.

## 📝 예시

이미지 파일을 추가한 후:

1. `public/images/projects/` 폴더에 이미지 파일 복사
2. 파일명 확인
3. 개발 서버 재시작 (`npm run dev`)
4. 브라우저에서 확인

이미지가 표시되지 않으면:
- 파일 경로가 정확한지 확인
- 파일명과 확장자가 일치하는지 확인
- 브라우저 콘솔에서 오류 메시지 확인
